La classe complex
En Python, on peut manipuler les nombres complexes directement :
>>> z=1+2j
>>> z
(1+2j)
>>> type(z)
<class ’complex ’>
Dans cet exemple, la variable z est un objet qui appartient `a la classe
complex (on dit aussi que z est une instance de la classe complex).
Une classe est donc un“moule `a objets”, qui d´efinit ce qu’on peut faire
avec les objets qui seront fabriqu´es avec ce moule.
Exemple : la classe complex d´efinit, pour chaque instance, les attributs
real et imag :
>>> z.real

>>> z.imag
2.0

Les classes en g´en´eral
De mani`ere g´en´erale, une classe peut d´efinir :
• les attributs des objets qu’elle permettra de d´efinir ;
• les m´ethodes associ´ees `a ces objets ;
• comment les op´erateurs (+ - * / // ** etc.) et fonctions standard

L’int´erˆet d’une classe est qu’elle permet l’encapsulation de toutes ces
notions, qui sont ainsi bien s´epar´ees des autres classes.
En python, tous les types sont des classes, et toutes les variables sont
