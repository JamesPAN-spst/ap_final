3. 第 1 章 — Python 类型补充

3.1 int:任意精度整数

  • 没有位数上限,2**1000 直接可写。
  • 但是 float(2**1000) 会 OverflowError(浮点数有上限)。
  • 进制字面量:0b…(二)、0o…(八)、0x…(十六);转换:bin / oct / hex / int(s, base)。

3.2 float:IEEE 754 双精度(64 位)

  • 大约 16 位有效数字;最大约 1.8e308,最小正数约 5e-324。
  • 超过上限 → inf;小于下限 → 0.0(下溢 sous-flux)。
  • 浮点加法不结合:1 + 1e-16 - 1 = 0.0 ⚠️
  • 陷阱:0.1 + 0.2 == 0.3 是 False。

3.3 complex:复数

z = 1 + 2j
z.real, z.imag, abs(z)         # 1.0, 2.0, sqrt(5)
import cmath
cmath.phase(z)                 # 辐角 = arctan(2)
复数不能比较大小(z1 < z2 报错),只能比较模长 abs(z)。

🍳 反转字典 (inverse d'un dict) — 高频考点
原字典 D : A → B,要返回 B → list(A)(因为多个键可能映射到同一个值): 
def inverse(D):
    R = {}
    for k, v in D.items():
        R.setdefault(v, []).append(k)
    return R
关键 API:setdefault(v, []) 既保证键存在,又返回该键对应的列表。

4. 第 2 章 — 函数、递归、生成器

4.1 函数定义的 4 条铁律

  1. 没写 return = 返回 None。
  2. 多返回值其实是元组:return a, b ⇒ 调用方可解包 x, y = f()。
  3. 定义签名时,无默认值参数必须在有默认值参数之前。
  4. 调用时,位置参数必须在关键字参数之前。

4.2 lambda 表达式

f = lambda x, y: x*y + 1            # 只能是单个表达式
sorted(L, key=lambda p: p[1])       # 按二元组的第二项排序
  • 不能写 return、不能写 if/for 语句块。
  • 可以用三元表达式:lambda x: 'A' if x > 0 else 'B'。

4.3 递归通用模板

def f(n):
    assert n >= 0          # 防御性断言
    if n == 0:              # 基准情形 (cas de base)
        return ...
    return ... f(n-1) ...   # 递推

4.4 记忆化 (mémoïsation)

D = {0: 0, 1: 1}
def fib(n):
    if n not in D:
        D[n] = fib(n-1) + fib(n-2)
    return D[n]
# 由 O(φⁿ) 优化到 O(n)
也可以用 functools.lru_cache 装饰器。

4.5 分治 (diviser pour régner) 三大经典

算法核心思想复杂度
快速排序 (Quicksort)选枢轴,分成 < 与 ≥ 两段递归平均 O(n log n),最坏 O(n²)
归并排序 (Tri fusion)对半切,递归排,再合并O(n log n)(最坏)
快速幂 (Exponentiation rapide)n 偶 → 平方;n 奇 → ×底数O(log n) 次乘法
🍳 快速排序短版(必背)
def tri(L):
    if not L:
        return []
    x = L[0]                                  # 枢轴
    L1 = [a for a in L[1:] if a <  x]
    L2 = [a for a in L[1:] if a >= x]
    return tri(L1) + [x] + tri(L2)
🍳 快速幂(可对数,也可对矩阵用)
def power(A, n):
    if n == 0: return 1            # 或单位元
    if n == 1: return A
    if n % 2:  return power(A, n - 1) * A
    B = power(A, n // 2)
    return B * B
例: 算 A⁴⁵,递归路径 45→44→22→11→10→5→4→2 ⇒ 共 8 次乘法。

4.6 生成器 (générateur, yield)

def pairs(L):                # 输入 list,产出偶数
    for x in L:
        if x % 2 == 0:
            yield x

list(pairs([1, 2, 3, 4]))                          # [2, 4]
sum(x*x for x in range(10**7) if x % 2 == 0)       # 内存友好
  • 函数中含 yield 即为生成器函数;调用时不会立刻执行函数体,而是返回生成器对象。
  • 每次 next(g) 才推进一步;遍历完抛 StopIteration。
  • yield from g2:把另一个生成器的产出"转发"出去。
  • 只能消费一次。要重新遍历得重新创建。
🍳 用生成器枚举所有子集 (parties d'un ensemble)
def parties(E):
    E = list(E)
    if not E:
        yield set()
    else:
        x = E[0]
        reste = set(E[1:])
        for S in parties(reste):
            yield S            # 不含 x 的子集
            yield S | {x}      # 含 x 的子集

5. 第 3 章 — 抽象数据类型

5.1 伪代码 (pseudo-code) 书写规范

  • 赋值用 ←,注释用 //。
  • 循环:pour tout x dans E(对所有)、tant que cond(当)、si … sinon(如果…否则)。
  • 必须写签名(每题失分点!),格式:
    fonction f(A, n) // arbre × entier → entier
  • 禁止用 Python 语法(伪代码不是 Python!)。len(L) 应写 cardinal(L) 或 longueur(L)。

5.2 队列 (FIFO,file)

def file_vide():            return []
def ajouter_element(x, F):  F.append(x)         # 入队尾
def extraire_element(F):    return F.pop(0)     # 出队头  ⚠️ O(n)
def premier_element(F):     return F[0]
def est_vide(F):            return len(F) == 0
典型用途: BFS(广度优先)、排队系统、按层遍历树。

5.3 栈 (LIFO,pile)

def pile_vide():    return []
def empiler(P, x):  P.append(x)
def depiler(P):     return P.pop()              # O(1)
def sommet(P):      return P[-1]
典型用途: 括号匹配、表达式求值、DFS(深度优先)、撤销操作。
🍳 括号匹配 (bien parenthésé) — 用栈
def bien_parenthese(s):
    P = []
    pairs = {')': '(', ']': '['}
    for c in s:
        if c in '([':
            P.append(c)
        elif c in ')]':
            if not P or P.pop() != pairs[c]:
                return False
    return len(P) == 0
思路:左括号入栈;遇右括号弹栈检查是否匹配;最后栈空才合法。

5.4 树 (arbre) — 用嵌套列表表示

A = ['x', ['a'], ['b', ['c'], ['d']]]   # 根 'x',下挂 2 棵子树

def etiquette(N): return N[0]            # 节点标签
def enfants(N):   return N[1:]           # 子树列表
def taille(N):    return 1 + sum(taille(E) for E in N[1:])
def hauteur(N):
    if len(N) == 1: return 1             # 叶子高度 = 1(本课程约定)
    return 1 + max(hauteur(E) for E in N[1:])
💡 树上递归的通用骨架
def f(N):
    # 1) 基准情形:叶子 (len(N) == 1)
    # 2) 递归处理每棵子树
    # 3) 合并子树的结果,再结合当前节点
    return combine([f(E) for E in N[1:]], etiquette(N))
常见考题变种:
  • 统计有 2 个孩子的节点数
  • 统计"至少有一个孩子是叶子"的节点数
  • 统计"标签与父节点相同"的节点数(把父节点作为额外参数传进去)
  • 每个节点的"得分" = 它孩子的标签之和,求最大值

5.5 图 (graphe) — 用 dict 套 set

G = {'A': {'B','C'}, 'B': {'A','D'}, ...}    # 无向图 = 对称邻接

def voisins(G, s):  return G[s]
def sommets(G):     return list(G.keys())
def ajouter_arete(G, s, t):
    G[s].add(t); G[t].add(s)
def supprimer_arete(G, s, t):
    G[s].discard(t); G[t].discard(s)

5.6 图的两种遍历

# DFS 递归版
def explore(G, s, M):
    M.add(s); print(s)
    for t in G[s]:
        if t not in M:
            explore(G, t, M)

# BFS 迭代版,顺便算最短距离
def distance(G, x, y):
    F = [x]; M = set(); d = 0
    while F:
        suiv = []
        for s in F:
            if s == y: return d
            if s not in M:
                M.add(s)
                for t in G[s]:
                    if t not in M:
                        suiv.append(t)
        d += 1
        F = suiv
    return None

5.7 欧拉定理 (Théorème d'Euler) ⭐

连通图存在欧拉通路(每条边恰好走一次) ⇔ 奇度数顶点个数为 0(欧拉回路)或 2(开放路径,起止于这两点)。 
def chemin_eulerien_existe(G):
    if not est_connexe(G):
        return False
    impairs = sum(1 for s in G if len(G[s]) % 2 == 1)
    return impairs in (0, 2)

6. 第 4 章 — 类与对象

6.1 类的解剖图

class Vecteur2d():
    def __init__(self, x, y):     # 构造器 (constructeur)
        self.x = x
        self.y = y
    def norme(self):              # 普通方法
        return (self.x**2 + self.y**2) ** 0.5
    def __repr__(self):           # 控制 print/repr 输出
        return f'({self.x},{self.y})'
    def __add__(u, v):            # 重载 + 运算符
        return Vecteur2d(u.x + v.x, u.y + v.y)

u = Vecteur2d(1, 2)               # 自动调用 __init__
print(u + Vecteur2d(3, 4))        # 调用 __add__,再调用 __repr__
  • self 就是"当前对象",必须作为方法第一个形参(命名 u、P 也行,但建议用 self)。
  • 外部访问属性用 obj.x;类内部用 self.x。

6.3 真题模板

🍳 二次三项式 Trinome(2024 年期末)
原题:实现 aX² + bX + c 形式的多项式类,支持加法、求导、打印。 
class Trinome():
    def __init__(self, a, b, c):
        self.a, self.b, self.c = a, b, c
    def __add__(P, Q):
        return Trinome(P.a + Q.a, P.b + Q.b, P.c + Q.c)
    def derive(self):
        return Trinome(0, 2*self.a, self.b)        # 求导
    def __repr__(self):
        return f'{self.a}X² + {self.b}X + {self.c}'
🍳 定点小数 Decimal(2025 年期末)
约定:用两个整数 (x, p) 表示数 x · 10⁻ᵖ。 
class Decimal():
    def __init__(self, x, p):
        self.x, self.p = x, p
    def __mul__(P, Q):                       # (x₁·10⁻ᵖ¹) × (x₂·10⁻ᵖ²)
        return Decimal(P.x * Q.x, P.p + Q.p) # = (x₁x₂)·10⁻⁽ᵖ¹⁺ᵖ²⁾
    def int(self):
        return self.x // 10**self.p          # 整数部分
    def __repr__(self):
        s = str(self.x)
        if len(s) < self.p + 1:              # 不够长就左侧补 0
            s = '0' * (self.p + 1 - len(s)) + s
        return s[:len(s) - self.p] + '.' + s[len(s) - self.p:]
repr 的诀窍:先把整数转成字符串,左侧补 0 直到至少有 p+1 位,然后在倒数第 p 位前插入小数点。

6.4 类的常见坑

  • 类体里直接写可变属性 → 所有实例共享: 
    class Mauvais:
    enfants = []          # ⚠️ 所有实例共用同一个列表!
class Bon:
    def __init__(self):
        self.enfants = [] # ✅ 每个实例自己的
  • 构造器默认参数用可变对象:跟普通函数一样会共享,要写 =None 模式。
  • "私有"属性约定:self._x 是软约定(请勿直接访问);self.__x 会触发名称改写 (name mangling) 变成 _Cls__x。
  • __eq__ 与 __hash__ 配套:重写了 __eq__ 但没写 __hash__,对象就会变成不可哈希,放不进 set。

7. 考试套路 Recettes anti-examen

过去几年期末高频题型 + 答题模板。每种"套路"都对照一道实际题。

R1. 表达式追踪题(set / dict 求值)

方法:逐行执行,把中间结果写出来。
易错点:
  • {} 是空 dict,不是空 set
  • set - list 直接报错
  • d.get(k, def) vs d[k](后者缺键报错)
  • L[:] 只是浅拷贝

R2. 树上"数一数"类(伪代码)

fonction f(N) // arbre × … → entier
  si N est une feuille:
    retourner cas_base
  c ← 0
  pour tout E dans enfants(N):
    c ← c + f(E)
    si condition(E, N):     // 例:E 与 N 同标签
      c ← c + 1
  retourner c
诀窍:如果判断条件依赖父节点,把父节点作为额外参数传入,签名写成 f(N, parent)。

R7. BFS 求最短距离

def dist(G, x, y):
    F = [x]; M = set(); d = 0
    while F:
        suiv = []
        for s in F:
            if s == y: return d
            if s not in M:
                M.add(s)
                suiv.extend(t for t in G[s] if t not in M)
        d += 1; F = suiv
    return None    # 不连通
关键:每"层"算同一个距离 d,所以用双层循环(外层换层,内层处理本层节点)。

R8. 欧拉判定题

def euler(G):
    if not est_connexe(G): return None        # 不连通 → 不存在
    impairs = [s for s in G if len(G[s]) % 2 == 1]
    if len(impairs) == 0: return 'cycle'      # 欧拉回路
    if len(impairs) == 2: return 'chemin'     # 欧拉通路
    return None                               # 都不是
口诀:"先看连通,再数奇度数,0 或 2 才行"。
